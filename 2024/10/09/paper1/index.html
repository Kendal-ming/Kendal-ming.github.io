【文献阅读】Robust Control of a Vehicle Steer-by-Wire System Using Adaptive Sliding Mode

Kendal2024-10-092024-10-15

科研入门所读文献，导师曾经所写论文，给与博主学习，现试着分析论文思路与结构

论文结构

一.摘要

努力翻译：

摘要：这篇论文呈现了一种自适应的滑动控制方法论针对的是汽车线控转向系统。首先该系统建模成一种二阶系统，从转向电机输入的电压到前轮转向角度。简单地说，由于轮胎和地面接触引起的自我对齐扭矩和摩擦被视作外部的干扰作用在该系统。接下来，一个ASM控制器被设计给这个系统，它不仅可以处理参数不确定性在植物模型中，也可以很有效地估计自对齐扭矩地系数。这种稳定性可以证明在李雅普诺夫意义上和收集控制参数指南被给予时。最后，实验执行转向系统去分别沿着slalom路径和圆周路径在不同道路条件下。实验证明：所推测的ASM控制器可以实现更强的鲁棒性在多样的道路条件下，致使显著减少道路循迹误差，在传统的滑动模块控制器和线性H控制器的对比下。

关键词： ASM、鲁棒控制、自对齐扭矩、SbW系统

二.介绍（前言、背景）

2.1 重要性：

  • 作为一个新兴发展的技术，汽车线性转向技术一直扮演着一种重要的作用在汽车产业。该系统主要的想法是去使用机电执行器和电子控制系统取代传统的机械联系。不同的汽车组成部分构成不同的线控技术，例如线控转向，brake-by-wire.shift-by-wire,throttle-by-wire技术。作为线控驱动系统的一部分，SBw系统是一个汽车转向应用中的创新技术。它可以有很多显著的优势：
    • 转向柱的移动可以减轻驾驶人的受伤当发生很大程度的正面碰撞时。
    • 在SBw系统中，在转向轮和前轮之间的传统的机械联系和液压系统，被机电执行器和人机交互替代，这不仅降低重量，噪声，躁动和能量消耗，还可以增加转向掌控，例如汽车的稳定性和机动性得到提升。
    • 反馈马达通常耦合于转向轮，并且提供驾驶人模仿在前轮和路面之间的作用力的触觉反馈。

2.2 以往研究

  • 机械模型
  • 控制系统控制策略
  • 自适应控制：提高鲁棒性

三.模型建立

3.1 仪器及其用途：

  • steering motor：通过机械装置提供前轮正确扭矩
  • SbW system：PC端=MARLAB+Advantech PCI multifunction card
    • 收集传感器信号并转换成控制信号
  • motor servo driver：把控制信号转换成电流信号，以控制steering motor
  • angle sensor：测量旋转角度（小齿轮角度以控制前轮转向）+测量方向盘的旋转角度

3.2 引出模型

  • 方程描述模型
  • 解释符号
  • 给出各k的取值
  • 考虑的参数不确定性
  • 实验装置不足的弥补：通过双曲正切函数，计算转过角度的自对齐扭矩，并得到电压控制信号函数。时变参数考虑了各路面的不同情况，添加了白噪声量

四.控制设计

4.1 ASM Control：

  • equivalent control input
  • ：保证系统鲁棒性（来自参数的不确定性和库仑摩擦力）
  • ：补偿自对齐扭矩
  • 引理1：以上定义后，闭环系统循迹误差可以稳定到0
    • 证明：李雅普诺夫一阶导小于0
    • Remark1：不需要提前知道自对齐扭矩的信息就可以估算扭矩，符合道路条件在真实情况是未知的真实情形
    • Remark2：使用boundary layer technique弥补的不稳定性

4.2 自对齐扭矩参数的估计

  • 线性滤波的方式设置，通过拉普拉斯变换求解出扭矩的表达式，在理想状态下系统只取决于扭矩
  • 设计简化表达式，取标称变量为转动惯量，则可得到最理想情况下的扭矩大小。
  • 后取作为扰动量引入，参数确定时为0，不确定时在邻域内接近

4.3 选择控制参数

  • ：大，反应快、高精度，但会高频率放大引入噪声干扰
  • ：大，反应快，但引入干扰
  • ：大，higher bandwidth of the self-aligning torque，增大
  • ：大，更少的扰动，但损失精度
