【文献阅读】Robust Control of a Vehicle Steer-by-Wire System Using Adaptive Sliding Mode

Kendal2024-10-092024-10-15

科研入门所读文献，导师曾经所写论文，给与博主学习，现试着分析论文思路与结构

论文结构

一.摘要

来一波小翻译：

摘要：这篇论文呈现了一种自适应的滑动控制方法论针对的是汽车线控转向系统。首先该系统建模成一种二阶系统，从转向电机输入的电压到前轮转向角度。简单地说，由于轮胎和地面接触引起的自我对齐扭矩和摩擦被视作外部的干扰作用在该系统。接下来，一个ASM控制器被设计给这个系统，它不仅可以处理参数不确定性在植物模型中，也可以很有效地估计自对齐扭矩地系数。这种稳定性可以证明在李雅普诺夫意义上和收集控制参数指南被给予时。最后，实验执行转向系统去分别沿着slalom路径和圆周路径在不同道路条件下。实验证明：所推测的ASM控制器可以实现更强的鲁棒性在多样的道路条件下，致使显著减少道路循迹误差，在传统的滑动模块控制器和线性H控制器的对比下。

关键词： ASM、鲁棒控制、自对齐扭矩、SbW系统

二.介绍（前言、背景）

2.1 重要性：

  • 作为一个新兴发展的技术，汽车线性转向技术一直扮演着一种重要的作用在汽车产业。该系统主要的想法是去使用机电执行器和电子控制系统取代传统的机械联系。不同的汽车组成部分构成不同的线控技术，例如线控转向，brake-by-wire.shift-by-wire,throttle-by-wire技术。作为线控驱动系统的一部分，SBw系统是一个汽车转向应用中的创新技术。它可以有很多显著的优势：
    • 转向柱的移动可以减轻驾驶人的受伤当发生很大程度的正面碰撞时。
    • 在SBw系统中，在转向轮和前轮之间的传统的机械联系和液压系统，被机电执行器和人机交互替代，这不仅降低重量，噪声，躁动和能量消耗，还可以增加转向掌控，例如汽车的稳定性和机动性得到提升。
    • 反馈马达通常耦合于转向轮，并且提供驾驶人模仿在前轮和路面之间的作用力的触觉反馈。

2.2 以往研究

  • 机械模型
  • 控制系统控制策略
  • 自适应控制：提高鲁棒性

三.模型建立

3.1 仪器及其用途：

  • steering motor：通过机械装置提供前轮正确扭矩
  • SbW system：PC端=MARLAB+Advantech PCI multifunction card
    • 收集传感器信号并转换成控制信号
  • motor servo driver：把控制信号转换成电流信号，以控制steering motor
  • angle sensor：测量旋转角度（小齿轮角度以控制前轮转向）+测量方向盘的旋转角度

论文分析

背景知识

物理知识

  • 库伦摩擦力：是一种固定的摩擦力，与速度无关，只取决于接触表面的性质
  • 粘滞摩擦（阻尼）：与物体的速度成正比
  • 自对准力矩：是轮胎与地面接触时产生的一种扭矩。当车辆前轮偏离直行时，轮胎的侧滑和地面的摩擦会产生一种力矩。试图将前轮自动恢复到直行方向

控制知识：

  • 滑膜控制：是一种鲁棒控制，特别适用于处理具有不确定性和外部干扰的系统。它通过创建滑动面（Sliding Surface），使系统状态沿着该面移动到目标点，能快速收敛并具有强鲁棒性
  • 自适应控制：能够在线调整控制器参数，以应对系统参数的变化或未知环境

自适应滑膜控制

  • 滑模控制：
    • 滑模面：定义在状态空间中的超平面。系统状态被设计成在这个滑模面上移动，从而实现控制目标
    • 滑模控制：让系统的状态在滑模面上滑动。一旦系统的状态进入滑模面，就会继续在该面上运动，保持稳定
    • 选择滑模面：可以定义为：
      • 系统稳定处于滑模面上时，且x的一阶导也为0，即x不会再发生变化脱离滑模面。令滑模面函数一阶导为0，则x二阶导为0，一阶导衡为0
      • 为设计参数，控制系统的收敛速度：比较大时，对误差的反应更灵敏；平衡状态需要为0，对于较大值时，细小误差就会造成较大偏离，则平衡效果也更精准
    • 控制律设计：
      • 等效控制律：用来保持在滑模面上，确保状态变量不脱离滑模面。是基于理想状态控制的，使用的是标称量和无干扰的情况，在论文中，通过把公式9代入成公式10，即暗中确定了系统处于滑模面上。
      • u1：抵抗参数不确定性。符号函数，取决于s的符号，当s>0是可以控制s减小以达到0为平衡状态。在论文中，取K为各数值最大边界值，提高出现不确定参数造成误差后的收敛速度。，为一个线性反馈，提供连续的控制力，能减少系统抖振现象。
      • u2：补偿自对准力矩。通过估计的，再将其转换为一个控制量，输入模型里则可抵消力矩的作用
      • boundary layer technique：通过用连续函数代替不连续函数，以减少系统的抖振现象
        • 的权衡：较大时，函数更为平缓，可以减小抖振，但此时函数值在大范围内持续减小，即减小了出现误差时每次的调整程度，不能及时控制系统，则降低了跟踪精度
  • 自适应：
    • 适应增益：设置成线性滤波器：
      • 、是滤波器的参数，是拉普拉斯变换中的微分算子。
      • 该形式的滤波器可以看作是一个比例加上一个与误差导数成比例的项，它可以根据误差的变化率调整适应增益
    • 设置简化控制律：在没有参数不确定性的时候，找到理想状态下的自对准力矩的估计
    • 引入：处理不确定性和外界扰动
